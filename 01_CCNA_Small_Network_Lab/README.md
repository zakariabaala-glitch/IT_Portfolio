# Small Enterprise Network Lab

## Objective
Simulate a small company network with multiple VLANs, routers, and switches.

## Tools Used
- Cisco Packet Tracer
- CCNA concepts

## Steps
1. Configure VLANs and assign IP addresses
2. Set up OSPF routing
3. Apply ACLs to control traffic
4. Test connectivity
